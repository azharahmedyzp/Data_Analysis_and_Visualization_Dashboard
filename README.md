## Features

* **Upload Dataset**: Upload your CSV files and preview the data.
* **Data Summary**: View descriptive statistics, mean, median, and standard deviation for numeric columns.
* **Visualizations**: Generate multiple plot types:

  * Histogram
  * Bar Chart
  * Box Plot
  * Heatmap
  * Scatter Plot
  * Pairplot
* **Missing Data Handling**:

  * Overview of missing values
  * Drop missing rows
  * Fill missing numeric values with column means
* **Download Report**: Download a summary CSV report of descriptive statistics.
* **About Section**: Information about the project and its tools.

## Usage

Run the Streamlit app:

```bash
python -m streamlit run Data_Analysis_and_Visualization_Dashboard.py
```

1. **Upload Dataset**: Upload a CSV file to start analyzing.
2. **Data Summary**: Explore descriptive statistics of your dataset.
3. **Visualization**: Select a plot type and columns to visualize the data.
4. **Missing Data**: Inspect and handle missing values.
5. **Download Report**: Export the summary statistics as a CSV file.
